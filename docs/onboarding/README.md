## Main Technologies
Depending on your role in the project, you may need to familiarize yourself with different technologies used in the Astro project. Here are some of the key technologies:
- **VS Code**: A popular code editor used for development. [See VS Code Configuration](vs-code/README.md) for recommended extensions and setup.
- **Git and GitHub**: Version control system and platform for hosting code repositories.
- **Docker**: Containerization platform used to create, deploy, and run applications in containers.
- **Python**: Primary programming language used for backend development and data processing.
- **FastAPI**: Web framework for building APIs with Python.
- **JavaScript/TypeScript**: Languages used for frontend development.
- **Angular**: Frontend framework for building web applications.
