## Tools
- **Git**: Version control system to manage code repositories. If in windows, use Git for Windows.
- (Optional to avoid the above) **GitHub Desktop**: GUI application for managing Git repositories.
- **VS Code**: Recommended code editor for development.
- **Docker**: Containerization platform to run services locally. Requires configuration to install.
- **uv**: A Python package manager to manage virtual environments and dependencies. In Windows it requires Python to be installed first.

> Note: More specific setup instructions (or configurations) for each component can be found in their respective directories within this knowledge base or online.
