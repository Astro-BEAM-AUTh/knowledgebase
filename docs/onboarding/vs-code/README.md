## Overview
VS Code is a popular code editor developed by Microsoft that provides a wide range of features and extensions to enhance the development experience. 
It supports multiple programming languages and offers built-in Git integration, debugging capabilities, and customizable themes.

## Configuration
### Extensions
Install the recommended extensions listed below to enhance your development experience or use one of the default VS Code Profiles that include some of these extensions. eg for a Python project you can use the "Python" default profile but you'll first have to create it.

- "njpwerner.autodocstring" - Automatically generates docstrings for Python functions and classes.
- "ms-python.autopep8" - Formats Python code according to PEP 8 standards.
- "alefragnani.Bookmarks" - Allows you to bookmark specific lines of code for easy navigation.
- "ms-azuretools.vscode-containers" - Provides tools for working with Docker containers directly from VS Code.
- "ms-vscode-remote.remote-containers" - Enables development inside a containerized environment.
- "docker.docker" - Adds Docker support to VS Code, allowing you to manage containers and images.
- "mikestead.dotenv" - Adds support for .env files, useful for managing environment variables.
- "purocean.drawio-preview" - Allows you to view and edit draw.io diagrams directly in VS Code (Install only if you plan to use draw.io for diagrams).
- "tamasfe.even-better-toml" - Provides enhanced support for TOML files, useful for configuration files.
- "mhutchie.git-graph" - Visualizes Git repositories and their commit history.
- "github.vscode-github-actions" - Integrates GitHub Actions into VS Code for managing workflows. (Install only if you plan to work with GitHub Actions).
- "GitHub.vscode-pull-request-github" - Manage GitHub pull requests and issues directly from VS Code.
- "codezombiech.gitignore" - Provides .gitignore templates for various programming languages and frameworks.
- "nhoizey.gremlins" - Highlights invisible and non-unicode characters in your code.
- "ecmel.vscode-html-css" - Enhances HTML and CSS editing capabilities in VS Code.
- "ms-python.isort" - Automatically sorts Python imports according to PEP 8 standards.
- "wholroyd.jinja" - Adds support for Jinja templating language in VS Code. (Install only if you plan to work with Jinja templates).
- "ms-toolsai.jupyter" - Provides Jupyter Notebook support in VS Code for interactive coding and data analysis. (Install only if you plan to work with Jupyter Notebooks).
- "James-Yu.latex-workshop" - Comprehensive LaTeX support for VS Code. (Install only if you plan to work with LaTeX documents).
- "docsmsft.docs-yaml" - Adds support for YAML files, useful for configuration and documentation.
- "bierner.lit-html" - Provides support for HTML inside of JavaScript and TypeScript files.
- "ritwickdey.LiveServer" - Launches a local development server with live reload feature for static and dynamic pages. (Install only if you plan to work with static web development).
- "yzhang.markdown-all-in-one" - Enhances Markdown editing with various features like preview, table of contents, and shortcuts.
- "yzane.markdown-pdf" - Converts Markdown files to PDF format.
- "bierner.markdown-mermaid" - Adds support for Mermaid diagrams in Markdown files.
- "HookyQR.minify" - Minifies code files to reduce their size.
- "IBM.output-colorizer" - Colorizes terminal output for better readability.
- "christian-kohler.path-intellisense" - Provides autocompletion for file paths in your code.
- "esbenp.prettier-vscode" - Integrates Prettier code formatter into VS Code for consistent code styling.
- "inferrinizzard.prettier-sql-vscode" - Adds Prettier support for SQL files.
- "ms-python.python" - Official Python extension for VS Code, providing rich support for Python development.
- "ms-python.vscode-pylance" - Fast and feature-rich language support for Python in VS Code.
- "KevinRose.vsc-python-indent" - Automatically adjusts indentation in Python files.
- "mechatroner.rainbow-csv" - Highlights CSV and TSV files with different colors for better readability. (Install only if you plan to work with CSV/TSV files).
- "humao.rest-client" - Allows you to send HTTP requests and view responses directly in VS Code. (Install only if you plan to work with APIs).
- "charliermarsh.ruff" - Integrates Ruff linter into VS Code for Python code quality checks.
- "alexcvzz.vscode-sqlite" - Provides SQLite database management capabilities within VS Code.
- "meganrogge.template-string-converter" - Converts strings to template strings in JavaScript and TypeScript files. (Install only if you plan to work with JavaScript/TypeScript).
- "Gruntfuggly.todo-tree" - Highlights TODO comments in your code for easy tracking of tasks.
- "redhat.vscode-yaml" - Adds comprehensive YAML support, including validation and autocompletion.
